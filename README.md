# Electric Vehicle Market Analysis – AtliQ Motors

## Project Overview
This project analyzes the Electric Vehicle (EV) market in India to support AtliQ Motors’ EV and hybrid vehicle expansion strategy. The dashboard provides insights into sales growth, EV penetration, regional opportunities, and future projections.

## Business Objectives
- Analyze EV sales and penetration trends in India
- Identify high-growth states for market entry
- Support manufacturing location and expansion decisions
- Provide strategic insights for top management

## Tools & Technologies
- Power BI (Data Modeling, DAX, Interactive Dashboards)
- SQL (Data Preparation)
- Excel (Data Cleaning)

## Key Insights
- EV penetration is increasing steadily across major states
- Certain states show strong adoption due to policy support and infrastructure
- Growth trends indicate strong future demand for EVs

## Report Structure
- Executive Overview
- Market & Competition Analysis
- Regional Opportunity Analysis
- Growth Trends & Projections
- Strategic Analysis

## Files Included
- Power BI (EV_project challenge.pbix) file
- Dashboard screenshots
- SQL queries
